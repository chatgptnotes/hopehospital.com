This is the **Google Business Profile workflow** of the Hope Hospital SEO
operating system. GBP is high-impact for hospital local SEO. Draft posts and
review replies — for the user to publish manually.

## Steps
1. Read `seo-system/project-context/master-context.md` (hospital facts, schemes,
   tone, hard constraints).

## Output
- **3 GBP post drafts** (theme = `$ARGUMENTS` if given, else pick from priorities:
  Ayushman/PMJAY, a department, a health-awareness day). Each: a tight headline,
  2–3 sentence body, a clear CTA (call / book / directions), and a suggested image
  idea. ~1500-char limit.
- **Review-reply templates** — 3 short, warm, non-templated-sounding replies: one
  for a 5★, one for a neutral 3★, one for a critical 1–2★ (acknowledge, take it
  offline to the hospital contact — never argue, never share patient details).

## Rules
- **Only true facts** — no invented services, doctors, stats, or claims. Rating is
  3.9/415. **No Hindi.**
- Reviews involve real patients: be empathetic, **never disclose any patient or
  medical detail**, and route specifics to a private channel.
- Draft only — the user publishes to GBP manually. Save drafts under `seo-system/gbp/`.
